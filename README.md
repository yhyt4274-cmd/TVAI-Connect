# 💬 TVAI Connect

A modern real-time chat application with a clean UI, friend system, and live messaging powered by Supabase.

---

## 🚀 Overview

TVAI Connect is a web-based messaging platform that allows users to create accounts, add friends, and chat in real time.  
It is built with a focus on performance, simplicity, and modern UI design.

---

## ✨ Features

- 🔐 Authentication system (Supabase + Mock mode fallback)
- 👥 Friend system (send / accept requests)
- 💬 Real-time messaging
- 🟢 Online/offline status
- 📱 Fully responsive design (mobile + desktop)
- 🎨 Modern glassmorphism UI
- 🌌 Animated 3D background using Three.js
- ⚡ Fast and lightweight frontend

---

## 🧠 Tech Stack

- HTML5
- CSS3 / TailwindCSS
- JavaScript (ES Modules)
- Supabase (Auth + Database + Realtime)
- Three.js
